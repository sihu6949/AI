# Daeyang File Locker (Windows 11 GUI)

오프라인 상태에서도 안전하게 파일을 암호화하고 복호화할 수 있는 Windows 11 스타일의 GUI 프로그램입니다.

## 주요 기능

1. **파일 암호화**: AES-256-GCM 알고리즘을 사용한 안전한 파일 암호화 (확장자가 `.daeyang`으로 변환됨).
2. **윈도우 탐색기 통합**: 프로그램 최초 실행 시 `.daeyang` 확장자가 레지스트리에 자동 등록되어, 파일 더블클릭 시 자동으로 복호화 창이 열립니다.
3. **마스터 비밀번호 (`Daeyang`)**: 사용자가 기존 비밀번호를 잊어버린 경우에도 마스터 비밀번호인 `Daeyang`을 입력하여 안전하게 파일 복호화가 가능합니다.
4. **임의 확장자 수정 감지**: 암호화되지 않은 일반 파일의 확장자만 임의로 `.daeyang`으로 변경한 경우, 시그니처 체크를 통해 `"사용자가 수정한 파일입니다."` 경고를 출력하고 복호화를 거부합니다.
5. **다크 모드 지원**: Windows 11 테마와 어우러지는 다크 모드 GUI가 적용되었습니다.

## 설치 및 실행 방법

### 의존성 설치 (최초 1회, 인터넷 필요)
```bash
pip install cryptography
```

### 프로그램 실행 (오프라인 가능)

#### 1. 일반 GUI 실행 (암호화/복호화 대시보드)
```bash
pythonw file_locker.py
```
*(참고: `pythonw`를 사용하여 백그라운드 콘솔 창 없이 GUI 앱만 실행할 수 있습니다.)*

#### 2. 탐색기 더블클릭 연동
- `.daeyang` 파일을 더블클릭하면 팝업 형태의 퀵 복호화 창이 뜹니다.
- Windows 11에서 최초 실행 시 "이 파일을 어떻게 여시겠습니까?"라는 팝업이 뜨면, **Daeyang File Locker** (또는 Python)를 선택하고 **"항상"**을 선택해 주시면 탐색기 연동이 완료됩니다.
